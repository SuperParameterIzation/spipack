Detailed Description

Estimate a weighted Laplacian operator using a graph Laplacian.

Let \(\psi\) be a probability distribution (with corresponding probability density function). Define the weighted Laplacian operator (given some function \(H\))

\begin{equation*} \nabla_{\psi}^2 H = \psi^{-1} \nabla \cdot (\psi \nabla f). \end{equation*}

This class uses \(n\) samples \(\{\boldsymbol{x}^{(i)}\}_{i=1}^{n}\) such that \(\boldsymbol{x}^{(i)} \sim \psi\) to approximate the action of this operator and to solve the weighted Poisson equation (given the right hand side \(R\))

\begin{equation*} \nabla_{\psi}^2 H = R \end{equation*}

such that \(\mathbb{E}_{\psi}[H] = 0\). Let \(I(i,j)\) be the index of the \(j^{th}\) furthest point from the sample \(\boldsymbol{x}^{(i)}\) (note that \(I(i,0)=i\)) and let \(\{ \boldsymbol{x}^{(I(i,j))} \}_{j=0}^{k}\) be the \(k\) nearest neighbors. For each sample, define the squared bandwith parameter

\begin{equation*} r_i^2 = \max_{j \in [1,k]}{\left( \|\boldsymbol{x}^{(i)} - \boldsymbol{x}^{(I(i,j))} \|^2 \right)} \end{equation*}

Define a compact kernel function \(k: \mathbb{R}^{+} \mapsto \mathbb{R}^{+}\) such that \(k(\theta)\) is decreasing and \(k(\theta) = 0\) if \(\theta \notin [0,1]\) (i.e., the support of the kernel is \([0,1]\)). Let

\begin{equation*} k_{l}(\boldsymbol{x}^{(i)}, \boldsymbol{x}^{(I(i,j))}) = k\left( \frac{\| \boldsymbol{x}^{(i)} - \boldsymbol{x}^{(I(i,j))} \|^2}{2^l r_i r_j} \right) \end{equation*}

Define the matrix \(\boldsymbol{K}\) such that the \((i^{th}, j^{th})\) component is

\begin{equation*} K_{ij} = \frac{k(h^{-2} \|\boldsymbol{x}^{(i)} -\boldsymbol{x}^{(i)}\|^2)}{\sqrt{d^{(i)} d^{(j)}}}, \end{equation*}

where \(h\) is the bandwidth parameter and \(d^{(i)} = \sum_{j=1}^{n} k(h^{-2} \|\boldsymbol{x}^{(i)} -\boldsymbol{x}^{(i)}\|^2)\). Additionally, define a diagonal matrix \(\boldsymbol{D}\) such that \(D_{ii} = \sum_{j=1}^{n} K_{ij}\). Define the heat matrix and discrete weighted Laplacian

\begin{equation*} \begin{array}{ccc} \boldsymbol{P} = \boldsymbol{D}^{-1} \boldsymbol{K} & \mbox{and} & \widetilde{\nabla}_{\psi}^2 = h^{-2} (\boldsymbol{I}-\boldsymbol{P}), \end{array} \end{equation*}

where \(\boldsymbol{I}\) is the identity matrix.

Evaluate kernel at neighbors.

Given a point \(\boldsymbol{x}\) and its nearest neighbors \(\{\boldsymbol{x}^{(j)}\}_{j=1}^{k}\), compute the kernel function \(k_h(\|\boldsymbol{x} - \boldsymbol{x}^{(j)}\|^2) = k(h^{-2} \|\boldsymbol{x} - \boldsymbol{x}^{(j)}\|^2)\).

Parameters
[in]xThe given point \(\boldsymbol{x}\)
[in]h2The squared bandwidth \(h^2\) that defines the kernel.
[in,out]neighborsA vector of the nearest neighbors. First: the neighbor's index, Second: (input) the squared distance ( \(\boldsymbol{x} \cdot \boldsymbol{x}^{(j)}\)) between the point \(\boldsymbol{x}\) and the neighbor, (output) the kernel evaluation \(k(\boldsymbol{x}, \boldsymbol{x}^{(j)})\)
Returns
The sum of the kernel evaluations \(d(\boldsymbol{x}) = \sum_{j=1}^{k} k_h(\|\boldsymbol{x}-\boldsymbol{x}^{(j)}\|^2)\)

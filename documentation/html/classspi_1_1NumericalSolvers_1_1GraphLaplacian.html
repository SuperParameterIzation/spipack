Detailed Description

Estimate a weighted Laplacian operator using a graph Laplacian.

Let \(\psi\) be a probability distribution (with corresponding probability density function). Define the weighted Laplacian operator (given some function \(H\))

\begin{equation*} \nabla_{\psi}^2 H = \psi^{-1} \nabla \cdot (\psi \nabla f). \end{equation*}

This class uses \(n\) samples \(\{\boldsymbol{x}^{(i)}\}_{i=1}^{n}\) such that \(\boldsymbol{x}^{(i)} \sim \psi\) to approximate the action of this operator and to solve the weighted Poisson equation (given the right hand side \(R\))

\begin{equation*} \nabla_{\psi}^2 H = R \end{equation*}

such that \(\mathbb{E}_{\psi}[H] = 0\). Configuration Parameters:

Parameter Key Type Default Value Description
"NumSamples" std::size_t - In the case where a random variable is passed to the constructor, we draw \(n\) samples from the distribution.
"MaxLeaf" std::size_t 10 The maximum leaf size for the kd tree (nanoflann parameter).

Find the points that are within radius \(r\) of a given point \(\boldsymbol{x}\).

Find all of the points \(\{\boldsymbol{x}^{(j)}\}_{j=1}^{k} \subseteq \{\boldsymbol{x}^{(i)}\}_{i=1}^{n}\) such that \(\|\boldsymbol{x}-\boldsymbol{x}^{(j)}\| \leq r\), where \(r>0\) is a given radius.

[in]rThe search radius \(r\)
[out]neighborsA vector of the nearest neighbors. First: the neighbor's index, Second: the squared distance ( \(d^2 = \boldsymbol{x} \cdot \boldsymbol{x}^{(j)}\)) between the point \(\boldsymbol{x}\) and the neighbor

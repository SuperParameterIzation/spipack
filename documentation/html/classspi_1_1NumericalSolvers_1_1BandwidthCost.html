Detailed Description

The cost function used to tune the bandwidth parameter \(\epsilon\).

If tune is true, we must choose the optimal \(\epsilon\) for the density estimation. Given the bandwidth exponent \(l \in [l_{min}, l_{max}]\), we choose \(\epsilon \in [2^{(l_{min})}, 2^{(l_{max})}]\). We do this by defining the parameters \(\Sigma_{l} = \frac{1}{n^2} \sum_{i,j=1}^{n} K_{\epsilon_l}^{(ij)}\) and \(\Sigma_{\epsilon_l}^{\prime} = (\log{(\Sigma_{\epsilon_{l+1}})}-\log{(\Sigma_{\epsilon_l})})/\log{(2)}\)

The optimal \(\epsilon\) maximizes \(\Sigma_{\epsilon}^{\prime}\) and the corresponding manifold dimension estimate is \(m = 2 \max{(\Sigma_{\epsilon}^{\prime})}\).

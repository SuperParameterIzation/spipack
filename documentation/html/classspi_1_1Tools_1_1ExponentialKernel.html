Detailed Description

An implementation of the exponential kernel.

The hat kernel is an isotropic kernel (spi::Tools::IsotropicKernel) such that

\begin{equation*} k(\theta) = a_0 \exp{(-a_1 \vert \theta \vert^{2p})} \end{equation*}

given parameters \(a_0, a_1>0\) and exponent \(p>0\).

Configuration Parameters:

Parameter Key Type Default Value Description
"Magnitude" double 1.0 The magnitude of the kernel (the parameter \(a_0\)).
"Scale" double 1.0 The parameter \(a_1\).
"Exponent" double 1.0 The value of the exponent (the parameter \(p\)).

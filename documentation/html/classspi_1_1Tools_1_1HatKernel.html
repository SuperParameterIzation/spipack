Detailed Description

An implementation of the hat kernel.

The hat kernel is a compact kernel (spi::Tools::CompactKernel) such that

\begin{equation*} k(\theta) = \begin{cases} a & \mbox{if } \theta \in [0,1] \\ 0 & \mbox{else} \end{cases} \end{equation*}

given the parameter \(a>0\).

Configuration Parameters:

Parameter Key Type Default Value Description
"Magnitude" double 1.0 The magnitude of the kernel (the parameter \(a\)).

Detailed Description

Discretely represent the Kolmogorov operator \(\mathcal{L}_{\psi,\alpha}\) using samples \(\{\boldsymbol{x}^{(i)}\}\) from the distribution \(\psi\).

Define the Kolmogorov operator applied to a smooth function \(f\)

\begin{equation*} \mathcal{L}_{\psi,\alpha} f = \Delta f + (2-2\alpha) \nabla f \cdot \frac{\nabla \psi}{\psi} \end{equation*}

Special cases:

  • \(\alpha=\frac{1}{2}\): The weighted Laplacian operator \(\mathcal{L}_{\psi,0.5} f = \Delta f + \nabla f \cdot \frac{\nabla \psi}{\psi} = \psi^{-1} \nabla \cdot (\psi \nabla f) = \Delta_{\psi} f\)
  • \(\alpha=1\): The Laplacian operator \(\mathcal{L}_{\psi,1} f = \Delta f\)

References:

  • "Variable bandwidth diffusion kernels" by T. Berry & J. Harlim
  • "Data-driven spectral decomposition and forecasting of ergodic dynamical systems" by D. Giannakis

Detailed Description

Find the nearest neighbors for points in sample collection.

Given \(\{\boldsymbol{x}^{(i)}\}_{i=1}^{n}\) samples from the distribution \(\psi\), this class finds the nearest neighbors \(\{\boldsymbol{x}^{(j)}\}_{j=1}^{k}\) to a given point \(\boldsymbol{x}\).

Configuration Parameters:

Parameter Key Type Default Value Description
"NumSamples" std::size_t - In the case where a random variable is passed to the constructor, we draw \(n\) samples from the distribution.
"MaxLeaf" std::size_t 10 The maximum leaf size for the kd tree (nanoflann parameter).
"Stride" std::size_t NumSamples/100 Build \(i \in [0, m]\) kd trees that ignore the first \(i d\) samples, where \(d\) is the stride parameter.
"NumThreads" std::size_t 1 The number of openMP threads available to this object.

) const

Find the squared bandwidth parameter for each sample.

The squared bandwidth is defined as \(r_i^2 = \frac{1}{k} \sum_{j=1}^{k} \| \boldsymbol{x}^{(i)}-\boldsymbol{x}^{(I(i,j))} \|^2\) for each sample, where \(I(i,j)\) is the index of the \(j^{th}\) closest neighbor—note that \(I(i,0) = i\).

Parameters
[in]kThe number of nearest neighbors ( \(k\))
[out]neighborsEach entry cooresponds to the vector nearest neighbors of the \(i^{th}\) sample. Each neighbor is stored as a pair. First: the index \(I(i,j)\) of the neighbor; second: the squared distance \(\| \boldsymbol{x}^{(i)}-\boldsymbol{x}^{(I(i,j))} \|^2\)
Returns
A vector such that the \(i^{th}\) entry is the bandwidth \(r_i\)

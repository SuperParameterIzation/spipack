Detailed Description

An implementation of an isotropic kernel.

An isotropic kernel takes the form of a decreasing function \(k:\mathbb{R}^{+} \mapsto \mathbb{R}^{+}\) such that \(k(\theta) = k(\boldsymbol{x}_1, \boldsymbol{x}_2) = k(\|\boldsymbol{x}_1 - \boldsymbol{x}_2\|^2)\).

Note: we define the kernel in terms \(\|\boldsymbol{x}_1 - \boldsymbol{x}_2\|^2\) (rather than \(\|\boldsymbol{x}_1 - \boldsymbol{x}_2\|\)) for numerical convenience.

Detailed Description

An implementation of a compact kernel.

A compact kernel takes the form of a decreasing function \(k:\mathbb{R}^{+} \mapsto \mathbb{R}^{+}\) such that \(k(\theta) = 0\) if \(\theta \notin [0,1]\) (i.e., the support of the kernel is \([0,1]\).)

Detailed Description

In implementation of the bump kernel.

The bump kernel is a compact kernel (spi::Tools::CompactKernel) such that

\begin{equation*} k(\theta) = \begin{cases} a_0 \exp{\left( a_1 \left( 1 - \frac{1}{1-\theta^p} \right) \right) } & \mbox{if } \theta \in [0,1) \\ 0 & \mbox{else} \end{cases} \end{equation*}

given the parameters \(a_0, a_1>0\) and the exponent \(p>0\).

Configuration Parameters:

Parameter Key Type Default Value Description
"Magnitude" double 1.0 The magnitude of the kernel (the parameter \(a_0\)).
"Scale" double 1.0 The scale of the kernel (the parameter \(a_1\)).
"Exponent" double 1.0 The exponent of the kernel (the parameter \(p\)).

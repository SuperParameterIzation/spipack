double SampleRepresentation::DefaultParameters::TruncationTolerance ( bool const  compact)
static

The truncation tolerance for the kernel matrix.

If the kernel is a compact kernel (spi::Tools::CompactKernel), return \(1\). Otherwise return \(-\log{(5 \times 10^{-2})}\).

This choice corresponds to truncting the exponential kernel when

\begin{equation*} \exp{\left(-\frac{\|\boldsymbol{x}-\boldsymbol{x^{\prime}}\|^2}{\theta}\right)} \leq 5 \times 10^{-2} \end{equation*}

Parameters
[in]compacttrue: This is a compact kernel; false: This is not a compact kernel
Returns
The default truncation tolerance parameter

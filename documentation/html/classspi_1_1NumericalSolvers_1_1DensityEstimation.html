Detailed Description

Estimate the density of a probability distribution \(\psi\) given samples \(\{\boldsymbol{x}^{(i)}\}_{i=1}^{n}\) such that \(\boldsymbol{x}^{(i)} \sim \psi\).

Let \(\boldsymbol{K}_{\epsilon}\) be the kernel matrix computing in spi::NumericalSolvers::SampleRepresentation::KernelMatrix using the bandwidth parameter \(r_i^2 = \frac{1}{k} \sum_{j=1}^{k} \| \boldsymbol{x}^{(i)}-\boldsymbol{x}^{(I(i,j))} \|^2\). We estimate the density at each sample as

\begin{equation*} \psi^{(i)}_{\epsilon} = \sum_{j=1}^{n} \frac{K_{\epsilon}^{(ij)}}{n (\pi \epsilon r_i^2)^{m/2}}, \end{equation*}

where \(m\) is the manifold dimension.

In addition to the parameters/options below, this class has the same parameters/options as spi::NumericalSolvers::SampleRepresentation.

Configuration Parameters:

Parameter Key Type Default Value Description
"BandwidthParameter" double 1.0 The parmeter \(\epsilon\) used to compute the kernel
"ManifoldDimension" double 2.0 The manifold dimension \(m\).
"TuneManifoldDimension" bool false Tune the manifold dimension \(m\); if we know the exact manifold dimension then we do not need to tune it.

References:

  • "Variable bandwidth diffusion kernels" by T. Berry & J. Harlim
  • "Data-driven spectral decomposition and forecasting of ergodic dynamical systems" by D. Giannakis

Eigen::VectorXd DensityEstimation::Estimate ( DensityEstimation::TuningData &  tune)

Estimate the density at each sample.

This function computes the bandwidth \(r_i^2 = \frac{1}{k} \sum_{j=1}^{k} \| \boldsymbol{x}^{(i)}-\boldsymbol{x}^{(I(i,j))} \|^2\) and the kernel marix \(\boldsymbol{K}_{\epsilon}\). We then estimate the density at each sample as

\begin{equation*} \psi^{(i)}_{\epsilon} = \sum_{j=1}^{n} \frac{K_{\epsilon}^{(ij)}}{n (\pi \epsilon r_i^2)^{m/2}}, \end{equation*}

where \(m\) is the manifold dimension.

We must choose the optimal \(\epsilon\) for the density estimation. Given the bandwidth exponent \(l \in [l_{min}, l_{max}]\), we choose \(\epsilon \in [\exp{(l_{min})}, \exp{(l_{max})}]\). We do this by defining the parameters \(\Sigma_{\epsilon} = \frac{1}{n^2} \sum_{i,j=1}^{n} K_{\epsilon}^{(ij)}\) and \(\Sigma_{\epsilon}^{\prime} = (\log{(\Sigma_{\epsilon+h})}-\log{(\Sigma_{\epsilon})})/(\log{(\epsilon+h)}-\log{(\epsilon)})\)

The optimal \(\epsilon\) maximizes \(\Sigma_{\epsilon}^{\prime}\) and the corresponding manifold dimension estimate is \(m = 2 \max{(\Sigma_{\epsilon}^{\prime})}\).

This function takes the candidate bandwidth exponents \(l\), and finds the corresponding \(\epsilon\) that maximizes \(\Sigma_{\epsilon}^{\prime}\). It the (re-)sets the bandwidth parameter (spi::NumericalSolvers::DensityEstimation::bandwidthPara) to the optimal value. If spi::NumericalSolvers::DensityEstimation::tuneManifoldDimension is true it will also reset the manifold dimension (spi::NumericalSolvers::DensityEstimation::manifoldDim).

This function does NOT compute the kd trees. It assumes that spi::NumericalSolvers::SampleRepresentation::BuildKDTrees has already been called.

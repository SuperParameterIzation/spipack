Detailed Description

Represent a distribution \(\psi\) using samples.

Let \(\psi\) be a probability distribution, which we represent here using \(n\) samples \(\{\boldsymbol{x}^{(i)}\}_{i=1}^{n}\) such that \(\boldsymbol{x}^{(i)} \sim \psi\).

Let \(r_i\) be a bandwidth parmeter associated with each sample. Define a compact kernel function \(k: \mathbb{R}^{+} \mapsto \mathbb{R}^{+}\) such that \(k(\theta)\) is decreasing and \(k(\theta) = 0\) if \(\theta > 1\) (i.e., the support of the kernel is \([0,1]\)). Let

\begin{equation*} k_{\epsilon}(\boldsymbol{x}^{(i)}, \boldsymbol{x}^{(j)}) = k\left( \frac{\| \boldsymbol{x}^{(i)} - \boldsymbol{x}^{(j)} \|^2}{\epsilon r_i r_j} \right). \end{equation*}

Define the kernel matrix \(\boldsymbol{K}_{\epsilon}\) such that the \((i,j)\) component is \(K_{\epsilon}^{(ij)} = k_{\epsilon}(\boldsymbol{x}^{(i)}, \boldsymbol{x}^{(j)})\).

Compute the average of the pair-wise kernel derivative evaluations \(\bar{k}^{\prime} = \frac{d}{d \epsilon} \left[ n^{-2} \sum_{i,j=0}^{n} k\left( \frac{\|\boldsymbol{x}^{(i)}-\boldsymbol{x}^{(j)}\|^2}{\epsilon r_i r_j} \right) \right] \).

We (recursively) compute the average

\begin{equation*} \bar{k}^{\prime} = \frac{d}{d \epsilon} \left[ n^{-2} \sum_{i,j=0}^{n} k\left( \frac{\|\boldsymbol{x}^{(i)}-\boldsymbol{x}^{(j)}\|^2}{\epsilon r_i r_j} \right) \right] = -n^{-2} \sum_{i,j=0}^{n} \left. \frac{d k}{d \theta} \right|_{\theta = \frac{\|\boldsymbol{x}^{(i)}-\boldsymbol{x}^{(j)}\|^2}{\epsilon r_i r_j}} \frac{\|\boldsymbol{x}^{(i)}-\boldsymbol{x}^{(j)}\|^2}{\epsilon^2 r_i r_j} \end{equation*}

Construct the (dense) kernel matrix \(\boldsymbol{K}_{\epsilon}\).

Let \(r_i^2 = \frac{1}{k} \sum_{j=1}^{k} \| \boldsymbol{x}^{(i)}-\boldsymbol{x}^{(I(i,j))} \|^2\) be the squared bandwidth associated with each sample such that \(I(i,j)\) is the index of the \(j^{th}\) furthest index from the point \(\boldsymbol{x}^{(i)}\). By default, this function defines the kernel matrix \(\boldsymbol{K}_{\epsilon}\) such that the \((i,j)\) component is

\begin{equation*} K_{\epsilon}^{(ij)} = k_{\epsilon}(\boldsymbol{x}^{(i)}, \boldsymbol{x}^{(j)}) = k\left( \frac{\| \boldsymbol{x}^{(i)} - \boldsymbol{x}^{(j)} \|^2}{\epsilon r_i r_j} \right). \end{equation*}

Recall that \(k(\theta)\) is the kernel function.

This function does NOT compute the kd trees. It assumes that spi::NumericalSolvers::SampleRepresentation::BuildKDTrees has already been called.

Compute the average of the pair-wise kernel second derivative evaluations \(\bar{k}^{\prime} = \frac{d^2}{d \epsilon^2} \left[ n^{-2} \sum_{i,j=0}^{n} k\left( \frac{\|\boldsymbol{x}^{(i)}-\boldsymbol{x}^{(j)}\|^2}{\epsilon r_i r_j} \right) \right] \).

Compute the average of the pair-wise kernel derivative evaluations in a given row.

We compute the average

\begin{equation*} \bar{k}_i^{\prime} = \frac{d}{d \epsilon} \left[ (2(m-k)-(i\in [m,k]))^{-1} \sum_{j=k}^{m} (2.0 - (j==i)) k\left( \frac{\|\boldsymbol{x}^{(i)}-\boldsymbol{x}^{(j)}\|^2}{\epsilon r_i r_j} \right) \right] = -(2(m-k)-(i\in [m,k]))^{-1} \sum_{j=k}^{m} (2.0 - (j==i)) \left. \frac{d k}{d \theta} \right|_{\theta = \frac{\|\boldsymbol{x}^{(i)}-\boldsymbol{x}^{(j)}\|^2}{\epsilon r_i r_j}} \frac{\|\boldsymbol{x}^{(i)}-\boldsymbol{x}^{(j)}\|^2}{\epsilon^2 r_i r_j} \end{equation*}

(note we can also compute the average second dervative—the formula is obtained by differentiating with respect to \(\epsilon\).)

At what tolerance should we truncate the kernel matrix?

Only used if truncateKernelMatrix is true.

When computing the kernel matrix \(K_{\epsilon}^{(ij)} = k_{\epsilon}\left( \frac{\| \boldsymbol{x}^{(i)} - \boldsymbol{x}^{(j)} \|^2}{\epsilon r_i r_j} \right)\) only fill the matrix when \(\frac{\| \boldsymbol{x}^{(i)} - \boldsymbol{x}^{(j)} \|^2}{\epsilon r_i r_j} < \alpha\). This is the parameter \(\alpha\).

If the kernel is a compact kernel (spi::Tools::CompactKernel), the default is \(\alpha = 1\). Otherwise the default is \(\alpha = -\log{(5 \times 10^{-2})}\).

This choice corresponds to truncting the exponential kernel when

\begin{equation*} \exp{\left(-\frac{\|\boldsymbol{x}^{(i)}-\boldsymbol{x^{(j)}}\|^2}{\epsilon r_i r_j}\right)} \leq 5 \times 10^{-2} \end{equation*}

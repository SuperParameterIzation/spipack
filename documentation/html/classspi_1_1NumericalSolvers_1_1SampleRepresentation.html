Detailed Description

Represent a distribution \(\psi\) using samples.

Let \(\psi\) be a probability distribution, which we represent here using \(n\) samples \(\{\boldsymbol{x}^{(i)}\}_{i=1}^{n}\) such that \(\boldsymbol{x}^{(i)} \sim \psi\).

Let \(r_i\) be a bandwidth parmeter associated with each sample. Define a compact kernel function \(k: \mathbb{R}^{+} \mapsto \mathbb{R}^{+}\) such that \(k(\theta)\) is decreasing and \(k(\theta) = 0\) if \(\theta > 1\) (i.e., the support of the kernel is \([0,1]\)). Let

\begin{equation*} k_{\epsilon}(\boldsymbol{x}^{(i)}, \boldsymbol{x}^{(j)}) = k\left( \frac{\| \boldsymbol{x}^{(i)} - \boldsymbol{x}^{(j)} \|^2}{\epsilon r_i r_j} \right). \end{equation*}

Define the kernel matrix \(\boldsymbol{K}_{\epsilon}\) such that the \((i,j)\) component is \(K_{\epsilon}^{(ij)} = k_{\epsilon}(\boldsymbol{x}^{(i)}, \boldsymbol{x}^{(j)})\).

Construct the kernel matrix \(\boldsymbol{K}_{\epsilon}\).

Let \(r_i^2 = \frac{1}{k} \sum_{j=1}^{k} \| \boldsymbol{x}^{(i)}-\boldsymbol{x}^{(I(i,j))} \|^2\) be the squared bandwidth associated with each sample such that \(I(i,j)\) is the index of the \(j^{th}\) furthest index from the point \(\boldsymbol{x}^{(i)}\). By default, this function defines the kernel matrix \(\boldsymbol{K}_{\epsilon}\) such that the \((i,j)\) component is

\begin{equation*} K_{\epsilon}^{(ij)} = k_{\epsilon}(\boldsymbol{x}^{(i)}, \boldsymbol{x}^{(j)}) = k\left( \frac{\| \boldsymbol{x}^{(i)} - \boldsymbol{x}^{(j)} \|^2}{\epsilon r_i r_j} \right). \end{equation*}

Recall that \(k(\theta)\) is the kernel function.

This function does NOT compute the kd trees. It assumes that spi::NumericalSolvers::SampleRepresentation::BuildKDTrees has already been called.

Note: children may override this and define the kernel using a different vector of bandwith parameters \(r_i\).
